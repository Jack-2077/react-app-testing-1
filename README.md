# React-app-testing-1
A React web app to try out React Testing Library and Jest

# About
It's a collection of my favourite themes from [MonkeyType](https://monkeytype.com/)

## Get started


```bash
npm init -y
npm i create-react-app
npx create-react-app my-app
cd my-app
npm start
```
